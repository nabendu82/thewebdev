Now, our mobile view is remaining but because of flexbox, most of it is already mobile friendly.

Add the below code in **style.css** for mobile view.

<div className={styles.code__block}>

```

 body {
  text-align: center;
 }
nav.flex {
  flex-direction: row;
 }
.header-img {
  margin-top: 40px;
  margin-left: 0;
  width: 100%;
 }
.flex {
  flex-direction: column;
 }
.box-reverse-mobile {
  flex-direction: column-reverse;
 }
footer ul {
  padding: 0;
  width: 100%;
 }
footer p {
  text-align: center;
 }
}
```

</div>
