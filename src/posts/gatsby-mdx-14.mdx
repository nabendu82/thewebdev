Welcome to part-14 of the series. In the last part we created menu for the Mobile view and in this part we will add functionality to it, to show the left or the right sidebar.

After a bit of research on the internet, i found [this](https://css-tricks.com/hamburger-menu-with-a-side-of-react-hooks-and-styled-components/) article which i will follow to create the menu.

The CSS-in-JS library of styled component is very much required for this part, so we will stop the development server and npm install it.

<div className={styles.code__block}>

```
    npm install --save styled-components
```

</div>

Next, we will create the **Burger** and **Menu** folders in **components** and the following files.

![Burger and Menu](https://ik.imagekit.io/thewebdev/thewebdev/14/1_dyloMCTmA-.png)*Burger and Menu*

Next, in the **Burger.js** file put the below content. It is used to show three horizontal lines, which forms our hamburger menu. We are also getting two props in this component.

![Burger.js](https://ik.imagekit.io/thewebdev/thewebdev/14/2_n59-3iBA8.png)*Burger.js*

Put the below styles in **Burger.styled.js** file. Here, we are hiding it at larger screen. We are also using **open** props for some transition, when it is clicked.

<div className={styles.code__block}>

```
    import styled from 'styled-components';

    export const StyledBurger = styled.button`
        position: absolute;
        top: 10%;
        left: 2rem;
        display: flex;
        flex-direction: column;
        justify-content: space-around;
        width: 2rem;
        height: 2rem;
        background: transparent;
        border: none;
        cursor: pointer;
        padding: 0;
        z-index: 200;

    &:focus {
            outline: none;
        }

    div {
            width: 2rem;
            height: 0.25rem;
            background: #0A0A0A;
            border-radius: 10px;
            transition: all 0.3s linear;
            position: relative;
            transform-origin: 1px;
            :first-child {
                transform: ${({ open }) => open ? 'rotate(45deg)' : 'rotate(0)'};
                }

    :nth-child(2) {
                opacity: ${({ open }) => open ? '0' : '1'};
                transform: ${({ open }) => open ? 'translateX(20px)' : 'translateX(0)'};
            }

    :nth-child(3) {
                transform: ${({ open }) => open ? 'rotate(-45deg)' : 'rotate(0)'};
            }

    @media(min-width: 992px) {
                display:none;
            }
    }
    `;

    export default StyledBurger;
```

</div>
