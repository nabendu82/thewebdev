Next, in **App.js** we will use **useState** to set **options**to **DEFAULT_OPTIONS**. We will also loop through the options and pass the name to individual **SidebarItem** element.

![App.js](https://ik.imagekit.io/thewebdev/React-projects/Photoshop-clone/15_VJm92BUSlK.png)*App.js*

Next, we will use that prop in our **SidebarItem.js** to show the name.

![SidebarItem.js](https://ik.imagekit.io/thewebdev/React-projects/Photoshop-clone/16_z7DJJ1_62m.png)*SidebarItem.js*

Next, back in **App.js** we will have another state variable **selectedIndex**, which is used to get the correct filter option through **selectedOption**. Next, we are also passing it to the **SidebarItem** component for active prop.

![App.js](https://ik.imagekit.io/thewebdev/React-projects/Photoshop-clone/17_KPhA1fVx3X.png)*App.js*

We are then going to use active in **SidebarItem**, where we are going to add **active** to className if it is passed.

![SidebarItem.js](https://ik.imagekit.io/thewebdev/React-projects/Photoshop-clone/18_3m2l8SdbIq.png)*SidebarItem.js*

We now want to change the active property depending on the click of button. For that we will add a handleClick function in the button’s onClick handler.

![SidebarItem.js](https://ik.imagekit.io/thewebdev/React-projects/Photoshop-clone/19_5kbvaxdDd.png)*SidebarItem.js*
