Welcome to a new React JS project, where we are going to build a Hangman app using React.

This project is based on the Vanilla JS project by Brad Traversy and the code for the same is [here](https://github.com/bradtraversy/vanillawebprojects/tree/master/hangman). We are basically going to convert it into a React project.
