Welcome to part-4 of the series. We will start from where we had left. We will do some coding now, but first start the project by moving to the directory and running gatsby develop command.

![gatsby develop](https://ik.imagekit.io/thewebdev/GatsbyTwilio/4/1_PYPx7Kxidc.png)*gatsby develop*

Now, if we go to [http://localhost:8000/](http://localhost:8000/) we will get the default page.

![Default Starter](https://ik.imagekit.io/thewebdev/GatsbyTwilio/4/2_ZozAZUNhth.png)*Default Starter*

Time to change the default starter, so open the **index.js** file.

![index.js](https://ik.imagekit.io/thewebdev/GatsbyTwilio/4/3_EzSWcHz_cQ.png)*index.js*

We will remove all the unnecessary things from here. Also, remove **page-2.js** file as we don’t need it.

![index.js](https://ik.imagekit.io/thewebdev/GatsbyTwilio/4/4_kDdCwiIHx.png)*index.js*

It’s also time to change some more things in this gatsby starter. Open the **layout.js** file in **components** folder and change the footer text.

![layout.js](https://ik.imagekit.io/thewebdev/GatsbyTwilio/4/5_HBOgVhmbB.png)*layout.js*

Also, i want to change the site title. So, head over to **gatsby-config.js** file and change it.

![gatsby-config.js](https://ik.imagekit.io/thewebdev/GatsbyTwilio/4/6_L_xA3gOeDE.png)*gatsby-config.js*

Now, when we go to [http://localhost:8000/](http://localhost:8000/) we will see the updated app.

![Updated](https://ik.imagekit.io/thewebdev/GatsbyTwilio/4/7_niQp2EaNL.png)*Updated*

Let’s create a simple functional based login form. Create a file **login-form.js** inside the **components** folder and put the below content in it. It is a simple form with a text field to enter the name and a button to submit the form.

![login-form.js](https://ik.imagekit.io/thewebdev/GatsbyTwilio/4/8_8mNHG4vA7H.png)*login-form.js*

Next, let’s show this component in our **index.js** file. So, import it and use it.

![index.js](https://ik.imagekit.io/thewebdev/GatsbyTwilio/4/9_1ENVuAkfJ.png)*index.js*

Our web-app now shows our not so beautiful login form.

![Login Form](https://ik.imagekit.io/thewebdev/GatsbyTwilio/4/10_puTnZnJnik.png)*Login Form*

![layout.css](https://ik.imagekit.io/thewebdev/GatsbyTwilio/4/11_jopmA0jCiZ.png)*layout.css*
