## **justify-items**

First we will look into **justify-items**, which align items along row axis. It can have any of the 4 values.

<div className={styles.code__block}>

```
    justify-items: start | end | center | stretch;
```

</div>

Let’s see **start** first. We will give all item a background of white and also, show our grid through Firefox developer edition.

<div className={styles.code__block}>

```
    .container {
            display: grid;
            grid-gap: 20px;
            grid-template-columns: repeat(10, 1fr);
            justify-items: start;
        }

        .itm{
            background: white;
        }
```

</div>

As, you can see it aligns items to the start edge of cell.

![**justify-items: start;**](https://ik.imagekit.io/thewebdev/CSS-Grid/7/2_xuwdOyAGqG.png)***justify-items: start;***

Now, **end** is opposite of start.

<div className={styles.code__block}>

```
    .container {
            display: grid;
            grid-gap: 20px;
            grid-template-columns: repeat(10, 1fr);
            justify-items: end;
        }

        .itm{
            background: white;
        }
```

</div>

It aligns items to the end edge of cell.

![**justify-items: end;**](https://ik.imagekit.io/thewebdev/CSS-Grid/7/3_TGX99UI4l.png)***justify-items: end;***

The **center** value will align items to center of cell.

<div className={styles.code__block}>

```
    .container {
            display: grid;
            grid-gap: 20px;
            grid-template-columns: repeat(10, 1fr);
            justify-items: center;
        }

        .itm{
            background: white;
        }
```

</div>

![**justify-items: center;**](https://ik.imagekit.io/thewebdev/CSS-Grid/7/4_c8YO0uGvZ6.png)***justify-items: center;***

The final value **stretch**, which is the default also. It will fill the whole width of the cell.

<div className={styles.code__block}>

```

            display: grid;
```

```
            align-items: stretch;
```
