Welcome to part-8 of the series. In this part we will start with our page transition, which will occur when we navigate to a different page.

For this we will be again using an awesome gatsby plugin and it is **gatsby-plugin-transition-link**. The doc link can be found [here](https://transitionlink.tylerbarnes.ca/docs/).

Now, as per the docs we need to npm install the plugin first. So, head over to your terminal and close **gatsby develop** if it is running. After that install the plugin through the terminal.

![npm install](https://ik.imagekit.io/thewebdev/GatsbyTourism/8/1_4zK5sxT0iq.png)*npm install*

Then open **gatsby-config.js** and add the plugin.

![Gatsby Config](https://ik.imagekit.io/thewebdev/GatsbyTourism/8/2_Pj321gCXFg.png)*Gatsby Config*

We also need to install one more package to start using Transition link. This package will be used for **Anilink**, which is a part of Transition link.

![Anilink](https://ik.imagekit.io/thewebdev/GatsbyTourism/8/3_iYx6gP_sT.png)*Anilink*

Then start the development server by **gatsby develop**. After that we will start replacing our **Link** with **AniLink**. So, head over to **Navbar.js** and update it. Notice, that we are using a prop paintDrip and a hex code. These are used to customize the transition. We can use other props also as per their [docs](https://transitionlink.tylerbarnes.ca/docs/anilink/).

![Navbar](https://ik.imagekit.io/thewebdev/GatsbyTourism/8/4_A2cTQJcgIt.png)*Navbar*

Next, hear over to **Footer.js** and update all **Links** to **AniLink**.

![Footer](https://ik.imagekit.io/thewebdev/GatsbyTourism/8/5_Ec54nTmF5.png)*Footer*

There is a Link in **index.js** also. So, let’s update it.

![index.js](https://ik.imagekit.io/thewebdev/GatsbyTourism/8/6_QiF_H3T91.png)*index.js*

Let’s also update the Link in **404.js** file.

![404.js](https://ik.imagekit.io/thewebdev/GatsbyTourism/8/7_hA89scFT1r.png)*404.js*

Now, if you move to any page by clicking on the link, you will get the below transition animation.

![Transition animation](https://ik.imagekit.io/thewebdev/GatsbyTourism/8/8_dI2TkxYCt.gif)*Transition animation*

Next, we will add Contact form to our project. So, head over to your code editor and add a file **contact.module.css** inside the **css** folder. The content for the same can be found from github [link](https://github.com/nabendu82/gatsbyTourism) for the project.

![contact.module.css](https://ik.imagekit.io/thewebdev/GatsbyTourism/8/9_DHYlBBcRPH.png)*contact.module.css*

Next, create a folder **Contact** inside **components** folder and file **Contact.js** inside it. It is a simple form with three fields for name, email and message.

<div className={styles.code__block}>

```
    import React from "react"
    import Title from "../Title"
    import styles from "../../css/contact.module.css"
    const Contact = () => {
        return (
            <section className={styles.contact}>
            <Title title="contact" subtitle="us" />
            <div className={styles.center}>
                <form className={styles.form}>
                <div>
                    <label htmlFor="name">name</label>
                    <input
                    type="text"
                    name="name"
                    id="name"
                    className={styles.formControl}
                    placeholder="john smith"
                    />
                </div>
                <div>
                    <label htmlFor="email">email</label>
                    <input
                    type="email"
                    name="email"
                    id="email"
                    className={styles.formControl}
                    placeholder="[email@email.com](mailto:email@email.com)"
                    />
                </div>
                <div>
                    <label htmlFor="message">message</label>
                    <textarea
                    name="message"
                    id="message"
                    rows="10"
                    className={styles.formControl}
                    placeholder="hello there"
                    />
                </div>
                <div>
                    <input
                    type="submit"
                    value="submit here"
                    className={styles.submit}
                    />
                </div>
                </form>
            </div>
            </section>
        )
    }

    export default Contact
```

</div>

Next, we will add this **Contact** component to the **contact.js** page.
