Welcome to the sixth part of Angular Basics. We will learn to change paths with Routing in this post. The starting point for this project can be taken from this zip file.

On running the project in localhost, we can find a simple project with different tabs. It is not working now and that is the thing, which we are going to fix.

![Basics](https://res.cloudinary.com/dxkxvfo2o/image/upload/v1660665704/New-blogs/Yashvendra_6/2_ansbsj.png)_Basics_

We have three components in our project which are the home, servers and users. In the **app.component.html** file, we are loading all of them.

![app.component.html](https://res.cloudinary.com/dxkxvfo2o/image/upload/v1660665704/New-blogs/Yashvendra_6/3_w8293e.png)_app.component.html_

To add routing in our project, we have to add the path in **app.module.ts** file, which will be mapped to the components.

![app.module.ts](https://res.cloudinary.com/dxkxvfo2o/image/upload/v1660665705/New-blogs/Yashvendra_6/4_ybew4f.png)_app.module.ts_

![app.component.html](https://res.cloudinary.com/dxkxvfo2o/image/upload/v1660665706/New-blogs/Yashvendra_6/5_hys92w.png)_app.component.html_

![localhost](https://res.cloudinary.com/dxkxvfo2o/image/upload/v1660665707/New-blogs/Yashvendra_6/6_lqldwd.png)_localhost_
